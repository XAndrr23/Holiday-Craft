# Holiday Craft

A short description of what this resource pack does:

## Minecraft Version
- Minecraft Bedrock Edition
- Tested on: 1.21.131+

## Features
- Custom grass block textures
- Converted from old MCPE textures

- No biome tinting on sides (by design)
